Maze Runner

Autonomous maze-solving robot using Embedded C

Project Overview

An autonomous robot that navigates through complex mazes using sensor fusion and pathfinding algorithms. The system combines infrared sensors, ultrasonic rangefinders, and wheel encoders to map and solve mazes in real-time.

Technology Stack

Embedded C Arduino PID Control A* Algorithm

Key Features

  • Real-time maze mapping with 5cm precision
  • Dead-end detection and path optimization
  • Adaptive motor control for different surfaces
  • LED matrix display showing current path

My Contributions

  • Developed the core pathfinding algorithm
  • Implemented sensor calibration routines
  • Optimized motor control parameters
  • Designed the PCB for sensor integration
