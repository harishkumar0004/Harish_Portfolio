YOLO Object Detection

Real-time object detection using YOLOv5

Project Overview

Implemented YOLOv5 model for real-time object detection capable of identifying 80+ object categories with high precision. The system processes video streams at 30 FPS with optimized performance for edge devices.

Technology Stack

Python PyTorch YOLOv5 OpenCV Flask

Key Features

  • Real-time detection at 30 FPS on NVIDIA Jetson
  • Custom trained on COCO dataset with 91% mAP
  • Web interface for remote monitoring
  • Optimized for low-power edge devices
  • Multi-class object classification
