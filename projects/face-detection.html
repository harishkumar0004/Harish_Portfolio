Unauthorized Face Detection

Security system using facial recognition

Project Overview

A security system that detects and alerts about unauthorized personnel using real-time facial recognition. The system compares detected faces against a database of authorized individuals and triggers alerts for unrecognized faces.

Technology Stack

Python OpenCV Face Recognition Flask

Key Features

  • Real-time face detection with 95% accuracy
  • Automatic email/SMS alerts
  • Multi-face detection in single frame
  • Low-light optimization
